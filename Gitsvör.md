 
cd - Skiptir hvaða directory þú ert í
ls - Sýnir hvaða skjöl eru til í þínu directory
pwd - prentar hvaða directory þú ert í
mkdir - býr til directory
 
git clone - afritar github repo
git status - sýnir hvaða stöðu þitt repo er í
git diff - sýnir breytingar á milli master repo og þínu

git log - sýnir commit history
git checkout - skiptir á milli branches
git branch - sýnir hvaða branches það eru

útgáfustýring - leyfir þér að sjá munin á milli commits
kostir við git - Leyfir fólk að vinna saman betur
